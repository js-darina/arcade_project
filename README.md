Проект *Hyperclimb*
Геймплей: Игрок управляет кубиком, который автоматически движется вверх. Задача — маневрировать между препятствиями, собирая внутриигровую валюту.
Экономика: В игре присутствуют различные виды монет (валюты) с разной ценностью.
Взаимодействие: На уровне встречаются подбираемые предметы, которые накладывают на игрока временные баффы (положительные эффекты) и дебаффы (отрицательные эффекты).
Прогрессия: Собранную валюту можно потратить в магазине.
Кастомизация: В магазине доступны скины, визуальные эффекты и другие косметические предметы для кубика.
